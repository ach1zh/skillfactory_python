# Практическая работа (PJ-1)

### Описание
Цель практической работы:
Разработать интерактивное веб-приложение с помощью Streamlit, которое позволяет анализировать и визуализировать исторические метеорологические данные.

Задача:
Реализовать приложение с использованием Streamlit, обладающее следующим функционалом:
1. Реализация календаря для выбора диапазона дат.
2. Вывод исходных данных:
- Отобразите таблицу с возможностью фильтрации (например, по городу или диапазону дат, если дата присутствует).
- Добавьте возможность сортировки и постраничного просмотра, если данных много.
3. Создание производных (новых) столбцов. На основе имеющихся данных добавьте полезные вычисляемые поля, например:
- Категория температуры ("холодно", "умеренно", "жарко").
- Уровень осадков ("без осадков", "небольшие", "сильные").
- Комфортность погоды (по вашей логике, например, на основе температуры и ветра).
4. Разведочный анализ данных (EDA):
- Постройте графики распределения признаков (temperature, precipitation, wind_speed): гистограммы, boxplots.
- Сравните погодные показатели между разными городами (столбчатые диаграммы, линейные графики и т. д.).
5. Временные ряды и прогнозирование (предполагается, что в данных есть временная метка — например, колонка date):
- Отобразите динамику температуры (или другого показателя) во времени.
- Реализуйте простой прогноз на основе скользящего среднего (например, окно в семь дней) и отобразите его на графике вместе с реальными данными.
6. Интерактивность:
- Используйте виджеты Streamlit (st.selectbox, st.multiselect, st.slider и другие), чтобы пользователь мог выбирать: город(а) для анализа; период времени (если есть дата); тип графика или метрики для отображения.

### Установка зависимостей
Для работы приложения необходимо подключить следующие библиотеки:
- streamlit (импортируется как st) — для создания веб-интерфейса и организации взаимодействия с пользователем;
- sqlite3 — для подключения к локальной базе данных SQLite и выполнения SQL-запросов;
- polars (импортируется как pl) — для быстрой загрузки и обработки больших объёмов данных;
- pandas (импортируется как pd) — для дополнительной обработки данных, особенно при работе с временными рядами и построении графиков;
- plotly.express (импортируется как px) — для построения интерактивных и наглядных графиков с минимальным количеством кода;
- datetime — для работы с датами и временем, включая форматирование и расчёты с временными интервалами.

### Запуск
Приложение реализовано в виде единого Python‑файла — для запуска достаточно выполнить его через Streamlit.
```streamlit run .\app.py```
