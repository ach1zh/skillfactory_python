# Практическое задание M31-L8

### Описание
Цель практической работы:
Научиться создавать приложение на Streamlit для анализа произвольного CSV-файла с табличными данными.

Задача:
Реализовать приложение с использованием Streamlit для анализа произвольного CSV-файла. Оно должно обладать следующим функционалом:
- загрузка CSV-файл для анализа;
- отображение содержимого файла в виде таблицы;
- поля для выбора столбцов столбцов таблицы;
- статистический анализ выбранного пользователем столбца: среднее значение, медиана, среднеквадратичное отклонение;
- построение графика для выбранных пользователем пар столбцов;
- построение разных типов графиков: графики линии, диаграмма рассеяния (пример);
- корректная обработка всех типов столбцов (числовые, текстовые, дата и время);

Дополнительно:
- кнопка для загрузки построенного графика;
- построение статистических графиков для выбранного пользователем столбца: график распределения в виде кривой или столбчатой диаграммы, график взаимного - распределения значений столбцов;
- использование состояния сеанса и кеширования.

### Установка зависимостей
Для работы приложения необходимо подключить следующие библиотеки:
- streamlit (импортируется как st) — для создания веб‑интерфейса;
- pandas (импортируется как pd) — для загрузки и обработки данных;
- plotly.graph_objects (импортируется как go) — для построения интерактивных графиков.

### Запуск
Приложение реализовано в виде единого Python‑файла — для запуска достаточно выполнить его через Streamlit.
```streamlit run .\app.py```
