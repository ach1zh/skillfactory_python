# Учебный проект курса Python-разработка платформы Skillfactory.

## Интерактивное веб-приложение для анализа метеоданных на Python: Streamlit + Airflow

### Описание учебного проекта
Цель проекта — Разработка интерактивного веб-приложения на основе Streamlit, предназначенного для анализа, визуализации и сравнения исторических и прогнозных метеоданных. Данные автоматически собираются из публичного API open-meteo.com с помощью Apache Airflow, оркестрируемого через Docker, и сохраняются в локальную базу данных SQLite (weather.db).

Проект реализует сквозной data-процесс: от сбора данных по расписанию до визуализации в пользовательском интерфейсе.

Приложение предоставляет пользователю инструменты для:
- фильтрации и просмотра данных,
- анализа погодных метрик,
- построения статистических графиков,
- оценки комфортности погоды,
- визуализации и сравнения прогнозов.

#### Запуск Airflow для автоматической загрузки данных
Для автоматического сбора и обновления метеоданных используется Apache Airflow, развёрнутый в Docker-контейнерах.
- /docker-compose.yaml — конфигурация контейнеров Airflow.
- /dockerfile - обеспечивает установку всех необходимых зависимостей в контейнер
- /dags/weather_dag.py — DAG, запускающий задачу загрузки данных каждый час.
- /scripts/fetch_and_store.py — скрипт, получающий данные по API и сохраняющий их в weather.db.
- /data/weather.db — файл базы данных с историческими и прогнозными метеоданными.

### Streamlit-приложение
- app.py — приложение реализовано в одном файле.

### Используемые библиотеки Python
- Streamlit — создание интерактивного веб-интерфейса.
- SQLite3 — работа с локальной базой данных.
- Polars — быстрая обработка больших объёмов данных.
- Pandas — анализ временных рядов и дополнительная агрегация.
- Plotly Express — построение интерактивных графиков.
- Airflow — оркестрация задач по расписанию.
- Requests — HTTP-запросы к API.
