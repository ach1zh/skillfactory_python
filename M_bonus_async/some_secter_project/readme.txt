main.py — точка входа в приложение

Отлично! Теперь, для того чтобы наш сервер мог обрабатывать запросы при помощи корутины index, 
её необходимо включить в роуты. Для этого мы создаём рядом (в той же папке) файл routes.py 
с процедурой, которая устанавливает связь между путём к запрашиваемой странице 
и соответствующим обработчиком. Но там у нас будет только один путь '/' и один обработчик index.
